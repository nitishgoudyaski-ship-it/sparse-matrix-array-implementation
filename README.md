# Sparse Matrix in C++ (Array Implementation)

This project demonstrates an array-based implementation of a Sparse Matrix using triplet representation (row, column, value).

## Example
Matrix (4x5) with non-zero elements:
- (0,1) = 10
- (1,3) = 20
- (2,2) = 30
- (3,0) = 40

### Triplet Form
Row Col Value  
0   1   10  
1   3   20  
2   2   30  
3   0   40
